Installation:
The application was tested on python v3.10.
Other requirements related to the installation are given in the requirements.txt file.

Running the application:
0. Add the openai api key in the file titled home.py and save the file.
1. Assuming that a virtual environment was created for the application activate the virtual environment.
2. run the python file by typing "python home.py"
3. Launch the web browser and type "http://127.0.0.1:500"
4. .\xflow_env\Scripts\activate
